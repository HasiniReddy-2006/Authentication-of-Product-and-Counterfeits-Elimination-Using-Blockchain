# Authentication of Product and Counterfeits Elimination Using Blockchain

## 📌 Project Overview

Counterfeit products are a major problem affecting consumers, manufacturers, and businesses. This project provides a blockchain-based solution to authenticate products and eliminate counterfeit goods by storing product information securely and transparently on a decentralized network.

The system allows manufacturers to register products on the blockchain, while customers can verify product authenticity using unique product information and barcode-based authentication.

---

## 🎯 Objectives

- Prevent counterfeit product circulation.
- Provide secure product verification.
- Maintain transparent and tamper-proof product records.
- Use blockchain technology for decentralized authentication.
- Build trust between manufacturers and customers.

---

## 🚀 Features

### Manufacturer Module
- Add new products to the blockchain.
- Store product details securely.
- Generate unique product identification records.

### Customer Module
- Scan/enter product details for verification.
- Check whether the product is original or counterfeit.
- Access blockchain-stored product information.

### Admin Module
- Manage product records.
- Monitor authentication requests.
